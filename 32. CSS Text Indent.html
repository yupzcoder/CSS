Text indent merupakan salah satu properti css dalam pengaturan tampilan atau style pada text. Fungsinya sebagai bagian penulisan awal pada sebuah paragraf. Nilainya bisa berupa persentase atau ukuran standar seperti px, cm, em.

Text indent ini menggunakan nilai dengan persentase. yang menginsyaratkan jarak text indent berdasar persentase screen atau layar

Text indent ini menggunakan nilai dengan satuan ukuran. lebih tepatnya disini menggunakan ukuran px.
