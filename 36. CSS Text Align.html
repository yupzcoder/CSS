Text align merupakan properti di dalam css, yang fungsinya mengatur letak atau posisi teks dalam sebuah teks dokument HTML. Setidaknya, terdapat 4 posisi standart yang sering digunakan dalam memposisikan dokumen HTML. Yaitu left, right center, dan justify.

Dokumen pada paragraf ini akan memposisikan sebagai rata kiri. Sehingga kita akan menggunakan value left pada properti text align di paragraf ini.

Dokumen pada paragraf ini akan memposisikan sebagai rata kanan. Sehingga kita akan menggunakan value right pada properti text align di paragraf ini.

Dokumen pada paragraf ini akan memposisikan sebagai rata tengah. Sehingga kita akan menggunakan value center pada properti text align di paragraf ini

Dokumen pada paragraf ini akan memposisikan sebagai rata tepi. Sehingga kita akan menggunakan value justify pada properti text align di paragraf ini.
