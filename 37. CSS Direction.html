Direction merupakan salah satu properti pada css. Fungsi dari properti yang satu ini sebagai bentuk arah tulisan dari sebuah teks dokumen HTML. Terdapat 2 jenis umum yang sering ditemui. Yaitu penulisan dari kiri ke kanan, atau kanan ke kiri. Biasa digunakan value ltr dan rtl

Dokumen ini akan menggunakan value direction dari kiri ke kanan. Yaitu value ltr

Dokumen ini akan menggunakan value direction dari kanan ke kiri. Yaitu value rtl
